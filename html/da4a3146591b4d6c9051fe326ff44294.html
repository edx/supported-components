In checkbox problems, learners select one or more options from a list of possible answers. To answer the problem correctly, a learner must select all of the options that are correct answers, and none of the options that are incorrect.

Authors have many configurable options that may affect the user experience. They can specify:

  • The number of attempts the learner has to answer the question.
  • Whether or not the learner has the ability to reset the problem.
  • Any number of hints the learner can choose to reveal.
  • An explanation of the answer
  • Feedback to be provided once the learner makes a submission
