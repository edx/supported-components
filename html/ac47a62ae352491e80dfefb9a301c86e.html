EdX allows authors to create content using either a limited WYSIWYG editor (Visual editor), or a Raw HTML editor.  When an author chooses the HTML Component button, they are presented with a list of 4 supported components:

  • Text
  • Announcement
  • iFrame
  • Raw HTML

The first three options are identical, with the exception of the default text that is present when the Visual editor is initialized. Authors have the full power and flexibility of HTML available to them when using the Raw HTML editor.  

Authors who use the WYSIWYG editor can create the following types of HTML elements:

  • Paragraphs
  • Headings (H3-H6)
  • Preformatted text blocks
  • Code blocks
  • Unordered lists
  • Ordered lists
  • Blockquotes
  • Hyperlinks
  • Images

Authors have the following text formatting options available to them when they are using the Visual editor:

  • Font
  • Bold
  • Italics
  • Underline
  • Text color
  • Nesting level (for lists)

Sample content

Examples of a Heading Level 3, paragraphs, and unordered lists are available above. Following are examples of other types of content and formatting you can create using the Visual editor. We will not include examples of everything you can do with the Raw HTML editor.  Course teams are responsible for ensuring the accessibility of the content they create using the Raw HTML editor.

Formatting

Authors can create bold, italic, and underlined text. They can even change the color of text, or combine all of the formats just mentioned.

Code blocks

You can use the Visual editor to easily author blocks of code:

# Prints out the numbers 0,1,2,3,4

for x in xrange(5): # or range(5)

    print x

Preformatted text

Or you can author preformatted text:

# Prints out the numbers 0,1,2,3,4
for x in xrange(5): # or range(5)
    print x
Lists

In addition to a bulleted (unordered) list, you can also create ordered lists:

  1. Do this first
  2. Do this next
  3. Then perform this final step
Nested lists

Lists can sometimes be more complicated. You can use the Visual editor's Increase and Decrease indent tools to created nested lists.

  1. Get in car
  2. Drive to the store
  3. Purchase:
    • Cheese
    • Eggs
    • Milk
  4. Return home
