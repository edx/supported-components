About This Course

Welcome to the edX demo test course! This course is designed to provide examples of all the course content types you can create using the edX platform. It includes examples of all the componenets and problem types found in our documentation, Building and Running an edX Course, as well as some features that are works in progress, experimental, or popular third party contributions.

The lastest version of this course is always available in the course's github repository. If you find something missing, feel free to open an issue.

Requirements

There are no additional requirements. Any feature included in this course should only require that a new value be added to the Advanced Module List under Advanced Settings.
